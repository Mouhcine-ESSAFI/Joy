# Joy Platform - Developer Documentation

This document provides a complete overview of the Joy Platform's frontend application, including setup instructions and a detailed data schema reference for developers.

## Getting Started

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.

## Data Schema

The data models for this application are defined in `src/lib/types.ts`. This file serves as the single source of truth for all data structures, ensuring consistency between the frontend application and the backend API.
